# 01_Core_Loop

## Overview
**Safety Code Violator** uses a tight, repeatable 15–30 minute session loop designed specifically for both PC and Android play sessions.

## The Loop (One Full Shift)

### 1. Briefing Phase (1–2 minutes)
- You receive a new **contract** from a shady client.
- Includes:
  - Location (Office Floor, Factory Line, Restaurant Kitchen, etc.)
  - Budget
  - Primary and optional objectives
  - Time limit (shift length)

**Example Contract:**
> “Renovate the 3rd floor open-plan office. Cut costs by 35%. Improve ‘team productivity’. Bonus if we can deny at least 3 insurance claims.”

### 2. Setup / Sabotage Phase (5–8 minutes)
- You have full control over the environment.
- Place hazards, modify equipment, adjust workflows, assign “maintenance tasks”, etc.
- Everything you do must look like legitimate (if incompetent) corporate decisions.
- This is the creative sandbox phase.

### 3. Simulation / Chaos Phase (8–15 minutes)
- Press “Start Shift”.
- Workers arrive and the simulation runs in **real-time**.
- You can:
  - Speed up time (1x / 4x / 16x / 64x)
  - Pause and intervene at any moment
  - Drop emergency hazards or “fixes”
- Watch chain reactions, funny worker behaviors, and escalating disasters.
- Manage the **Plausible Deniability Meter** in real time.

### 4. Debrief & Scoring Phase (2–3 minutes)
- Automatic highlight reel of the best (or most ridiculous) moments.
- Final statistics:
  - Injuries / Fatalities
  - Chaos Score
  - Plausibility Rating
  - Viral Impact
  - Insurance Money Saved (for the company)
- Unlock new hazards, locations, or traits based on performance.

---

## Session Length Control
- Average session: **15–25 minutes**
- Players can end a shift early if desired
- Fast-forward heavily during slow periods
- Designed so you can comfortably finish 1–2 full shifts per sitting on Android

---

## Progression Between Sessions
- After each shift you return to a central **Violator Hub** (your shady office).
- Review overall stats, unlock new tools/locations, accept next contract.
