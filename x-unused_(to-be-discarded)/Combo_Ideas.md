# Combo_Ideas

## Overview
This file collects fun, high-value hazard combinations that players will discover and perfect over time. These are the real “money shots” of the game.

## Classic Combos (Early Game)

### The Office Classics
- **The Kevin Special**  
  Wet floor + Faulty rolling chair + Heavy filing cabinet → Flying intern across the office

- **Coffee Apocalypse**  
  Faulty coffee machine + Overloaded power strip + Sleep-deprivation schedule → Multiple electrocutions + fire

- **Meeting Room Massacre**  
  Unstable projector mount + Wobbly chairs + "Team-building exercise" → Domino effect

### Factory Combos
- **Conveyor Carnage**  
  Faulty conveyor + Loose safety guard + Overworked employees → Multiple amputations / crushing

- **Forklift Fiasco**  
  Faulty brakes + Unsecured cargo + Distracted forklift driver → Chain pile-up

### Restaurant Combos
- **Grease Inferno**  
  Grease spill + Open flame + Rushing waiter → Fire + flying burning food

- **Deep Fryer Disaster**  
  Overheated fryer + Wet floor + "Helpful" coworker → Boiling oil shower

## Advanced / Showstopper Combos

- **Vertical Doom** (Construction Site)  
  Unstable scaffolding + Crane with faulty cable + Workers below → Multi-story collapse

- **Corporate Karma**  
  HR Karen + "Ergonomic" chair sabotage + Faulty monitor + Wet floor → Perfect ironic death

- **The "Totally Legitimate Renovation"**  
  Removed fire exits + Cheap flammable materials + Overloaded electrical + "Modern open concept" → Total building inferno

## Combo Scoring Bonuses
- **Length Bonus**: More hazards in one chain = higher multiplier
- **Thematic Bonus**: Deaths that match the location or worker personality
- **Plausibility Bonus**: Combo looks extremely believable
- **Creativity Bonus**: Using rarely combined hazards

## Design Goal
Encourage players to experiment and feel like evil geniuses when they discover new deadly synergies. The game should celebrate particularly beautiful combos with special slow-motion, sound effects, and newspaper headlines.
