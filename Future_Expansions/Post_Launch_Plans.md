# Post_Launch_Plans

## Post-Launch Roadmap

### Phase 1: Launch + First 3 Months
- Bug fixes and performance optimization (especially Android)
- Balance tweaks based on player feedback
- Steam achievements + leaderboards
- Basic Steam Workshop support for sharing highlight reels

### Phase 2: First Major Update (Free)
- New Location: **Chemical Processing Plant**
- 8–10 new hazards
- New worker personality pack (“Union Workers”)
- Improved replay system
- More contract templates

### Phase 3: Paid DLCs / Major Content Updates

#### DLC 1 – “Hospital Horror”
- Hospital location
- Medical-themed hazards and dark humor
- New objectives focused on “patient safety”

#### DLC 2 – “Entertainment Industry”
- Amusement Park + Cruise Ship
- Bigger crowds and vertical chaos
- Viral clip editor improvements

#### DLC 3 – “Executive Privilege”
- Luxury high-rise + Corporate HQ
- Richer targets, higher budgets, stricter investigators
- New “Executive” worker personality pack

### Long-Term / Stretch Goals
- **Modding Tools** – Full level editor + hazard creator
- **Multiplayer Chaos** – Co-op or versus mode (two Violators in the same building)
- **Endless Mode** – Procedurally generated mega-buildings
- **Space Station** – Ultimate endgame location
- **Mobile Exclusive Content** – Touch-optimized contracts or shorter shifts

## Monetization Strategy
- Base game: $9.99–$14.99
- DLCs: $4.99–$7.99 each
- Cosmetic packs (evil laughs, slow-mo filters, UI themes)
- Free updates for core improvements

## Community Goals
- Strong Steam Workshop integration
- Regular “Community Challenge” contracts
- Highlight reel sharing (Twitter/X, TikTok, YouTube integration)
- Player-created location contest winners added officially
